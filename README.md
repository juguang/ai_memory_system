# AI个人记忆系统

一个基于Claude Code和Obsidian的个人知识管理系统，通过4层记忆架构帮助你建立长期、可追溯的个人记忆。

## ✨ 特点

- **📦 零依赖**：不需要向量数据库，基于Claude Code和Obsidian
- **🔒 隐私安全**：所有数据存储在本地，完全掌控
- **🤖 自动化**：通过Claude Code skills自动记录和复盘
- **📊 4层架构**：从日常事件到核心价值观，层次清晰
- **🔄 持续进化**：通过周复盘、月复盘不断提炼认知

## 🎯 系统架构

```
L4 核心层 ← 只能手动填写（价值观、信念、身份）
    ↑
L3 认知层 ← 月复盘提炼（思维模式、决策框架）
    ↑
L2 行为层 ← 周复盘提炼（习惯、偏好）
    ↑
L1 情境层 ← 日常记录（事件、决策、情绪）
    ↑
Claude Code对话 + Obsidian文件
```

## 🚀 快速开始

### 环境要求
- [Obsidian](https://obsidian.md/) - 笔记管理
- [Claude Code](https://claude.ai/code) - AI编程助手

### 安装步骤

1. **复制文件到你的Obsidian vault**
   ```
   AI_MEMORY/ → 你的Obsidian库根目录/
   .claude/skills/mem-* → 你的Obsidian库/.claude/skills/
   ```

2. **填写L4核心层**
   - 打开 `AI_MEMORY/L4_核心层/核心价值观.md`
   - 打开 `AI_MEMORY/L4_核心层/人生信念.md`
   - 打开 `AI_MEMORY/L4_核心层/身份定义.md`
   - ⚠️ **只能手动填写，不要让AI自动生成！**

3. **开始使用**
   ```
   在Claude Code中说：
   - "记录到记忆系统：今天完成了XXX项目"
   - "周复盘"
   - "查询记忆：我的工作习惯是什么？"
   ```

## 📖 使用指南

### 日常记录
重要对话后，让Claude记录：
```
记录到记忆系统：今天决定采用React而不是Vue，
因为团队更熟悉，而且生态更成熟。
```

### 周复盘
每周日晚上说"周复盘"，Claude会：
1. 扫描本周修改的Obsidian文件
2. 分析L1记录，识别重复行为（出现3次以上）
3. 生成候选列表
4. 等待你确认后更新L2行为层

### 月复盘
每月最后一天说"月复盘"，提炼认知模式到L3

### 记忆查询
随时说"查询记忆：XXX"

## 💡 核心设计原则

### L4核心层保护机制
L4核心层（价值观、信念、身份）**只能手动修改**，Claude不能自动更新

### 渐进式提炼
```
日常记录 (L1)
    ↓ 出现3次+
提炼行为模式 (L2)
    ↓ 多个行为指向同一原则
提炼思维模式 (L3)
    ↓ 长期验证后
手动更新 (L4)
```

### 记录密度控制
聚焦：
- ✅ 重要决策、情绪波动、偏好表达、行为模式
- ❌ 琐碎日常、纯信息摘录

## ⚠️ 注意事项

1. **隐私安全**：记忆文件包含个人隐私，注意备份
2. **定期复盘**：每周日晚周复盘，每月最后一天月复盘
3. **L4谨慎**：L4核心层不要轻易修改
4. **数据备份**：建议通过iCloud/坚果云等多设备同步
