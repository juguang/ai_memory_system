# AI个人记忆系统 - 使用说明

## 🎯 系统目标

让Claude Code真正"认识"你：
- ✅ 记住你的价值观和偏好
- ✅ 理解你的工作习惯和思维方式
- ✅ 提供个性化的建议和帮助
- ✅ 像数字助手一样持续进化

---

## 🚀 快速开始

### 第一步：初始化L4核心层（必须手动）

**问自己3个问题**：
1. 我最看重什么？（核心价值观）
2. 我相信什么？（人生信念）
3. 我是谁？（身份定义）

把答案写入：
- `L4_核心层/核心价值观.md`
- `L4_核心层/人生信念.md`
- `L4_核心层/身份定义.md`

### 第二步：测试系统

```
问Claude："根据我的记忆，我通常如何处理复杂任务？"

Claude应该：
- 读取INDEX.md
- 检索L2_行为层
- 给出个性化回答
```

### 第三步：日常使用

```
每次重要对话后，说一句：
"把这次对话记到记忆系统"

Claude会自动提炼并更新
```

## ⏰ 日常流程

### 每天（0-3分钟）

**重要事件时**：
```
"把今天的讨论记到记忆系统"
```

**普通日子**：
```
什么都不用做 ✅
```

### 每周（5-10分钟）

**周日晚上**：
```
"周复盘"
```

Claude会：
- 分析本周L1记录
- 识别重复模式
- 建议提炼到L2

### 每月（15-20分钟）

**月末**：
```
"月复盘"
```

Claude会：
- 分析本月L2更新
- 提炼思维模式到L3
- 生成月度报告

### 每季度（30-60分钟）

**季度末**：
```
"季度复盘"
```

Claude会引导你：
- 深度反思价值观
- 手动更新L4核心层
- 重新校准全系统

## ⚠️ 重要规则

### 1. L4核心层只能手动修改

```
❌ Claude不能自动更新L4
✅ 你必须手动编辑L4文件
✅ Claude只能建议
```

### 2. 来源追溯

每个记忆都要标注来源：
```markdown
来源: [Daily/2025-12-27.md]
首次出现: 2025-12-27
出现次数: 3次
```

### 3. 定期回顾

- **L1→L2**: 每周自动建议
- **L2→L3**: 每月深度分析
- **L3→L4**: 每季度手动反思

---

## 🛠️ 故障排除

### 问题：Claude记不住我的偏好

**解决**：
1. 检查是否明确记录到L2
2. 检查PROMPTS.md是否加载
3. 提醒Claude："检索我的记忆系统"

### 问题：记忆太多，检索很慢

**解决**：
1. 定期清理L0临时数据
2. 归档旧的L1记录（超过30天）
3. 优化INDEX.md索引

### 问题：L4很久没更新

**解决**：
```
运行："季度复盘"
Claude会引导你反思
```

## 📊 系统健康度

### 好的状态：

- ✅ L4核心层：3个月内更新过
- ✅ L2行为层：10+个记录
- ✅ L1情境层：每周新增2-5条
- ✅ 每周做周复盘
- ✅ 每月做月复盘

### 需要关注：

- ⚠️ L4超过3个月未更新
- ⚠️ L1积累超过50条未复盘
- ⚠️ L2少于5个记录
- ⚠️ 超过2周未复盘

## 💡 最佳实践

### DO ✅

- 定期复盘（每周/每月）
- 明确表达偏好
- 主动触发记录
- 手动维护L4核心层
