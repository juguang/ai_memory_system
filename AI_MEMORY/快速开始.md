## 📝 3步开始使用

### 步骤1：填写L4核心层（必须）⭐⭐⭐

**为什么必须**：L4是整个系统的"校准锚点"，Claude会用这些价值观来校准所有回答。

**需要填写3个文件**：

#### 1.1 核心价值观.md

打开 `AI_MEMORY/L4_核心层/核心价值观.md`，回答：

```
我最看重什么？（举例）

价值观1：成长优先
- 含义：我更看重学习和成长，而非短期收益
- 案例：2025-01选择接受低薪但成长性高的工作
- 边界：除非影响基本生活，否则优先考虑成长

价值观2：真实性
- 含义：我重视诚实和透明
- 案例：即使在困难情况下也选择说实话
- 边界：不会伤害他人的前提下

价值观3：[你自己填写]
```

#### 1.2 人生信念.md

打开 `AI_MEMORY/L4_核心层/人生信念.md`，回答：

```
我相信什么？（举例）

关于世界：世界是可以通过努力改变的
关于人：大多数人本质是好的
关于自己：我可以学会任何我想学的东西
```

#### 1.3 身份定义.md

打开 `AI_MEMORY/L4_核心层/身份定义.md`，回答：

```
我是谁？（举例）

核心身份：
1. 终身学习者 - 我相信持续学习是人生的意义
2. 问题解决者 - 我享受面对和解决复杂问题
3. [你自己填写]
```

**预计时间**：15-30分钟

## ⚠️ 常见问题

### Q1：L4核心层太难填写怎么办？

**A**：可以先简单填写，后续随时更新。关键是开始，不一定要完美。

```
最小填写示例：
核心价值观：成长、真实、自由
人生信念：我可以改变、世界是公平的
身份定义：学习者、创造者
```

---

### Q2：我可以跳过L4，直接使用吗？

**A**：技术上可以，但**不推荐**。

- 没有L4，Claude无法基于你的价值观校准回答
- 系统效果会大打折扣
- 建议：至少花15分钟填写一个简化版

---

### Q3：多久更新一次L4？

**A**：
- **最小频率**：每季度回顾一次
- **推荐频率**：每月检查一次
- **触发时机**：当你的人生发生重大变化时

---

### Q4：其他文件需要现在填写吗？

**A**：不需要！

- L3、L2、L1会随着使用自然积累
- Intent层（目标、偏好、约束）可以慢慢填写
- 只有L4必须先填写

---

### Q5：系统会自动工作吗？

**A**：会的！

当你与Claude对话时，Claude会：
1. 自动读取 PROMPTS.md
2. 按协议执行检索和更新
3. 你只需要说"记录到记忆系统"即可
