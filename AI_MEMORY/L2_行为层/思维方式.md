# 思维方式

## 📋 系统性思维倾向

### 习惯描述
在处理新事物、新任务或复杂问题时，倾向于：

1. **建立结构化框架**
   - 优先考虑整体架构和系统设计
   - 关注各部分之间的关联性
   - 重视可扩展性和可维护性

2. **渐进式实施**
   - 从简单开始，逐步完善
   - 不追求一次到位，重视迭代优化
   - 基于现有工具和资源，避免过度设计

3. **文档化和规范化**
   - 重视流程和规范的建立
   - 喜欢记录和总结
   - 建立可复用的模式
